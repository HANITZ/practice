# [2023 토스 NEXT 개발자 챌린지] Frontend 직군 과제테스트

## Getting started

```sh
yarn install
yarn start
```

## Testing

아래 명령어로 더미 데이터에 기반한 구현을 테스트할 수 있습니다.

```sh
yarn test
```

키패드가 열리고 닫히는 상태
상태1: 오픈되어 있는 키패드 상태
상태2: 비밀번호 입력값 상태
상태3: 비밀번호 확인 입력값 상태

인풋 창이 포커스 되면 키패드가 나타난다.
포커스가 사라지면 키패드가 사라진다.
확인을 누르면 키패드가 닫힌다.
두개의 키패드가 다름.
